# 🚀 Post Web App

A full-stack web application where users can create, manage, and interact with posts.  
Built with authentication and engagement features like likes, this project reflects real-world backend and frontend integration.

---

## 📌 About The Project

Post Web App is designed to simulate how modern social/content platforms work.

Users can register, log in, create posts, and interact with other content through likes. The project focuses on building a structured backend,
handling user sessions, and managing dynamic data efficiently.

---

## ✨ Features

- 🔐 **User Authentication** (Signup / Login / Logout)  
- 📝 **Create new posts**  
- 👀 **View all posts**  
- ✏️ **Edit your posts**  
- ❌ **Delete posts**  
- ❤️ **Like / Unlike posts**  
- 👤 **User-specific content handling**  
- 📦 **Secure data storage (MongoDB)**  

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript, EJS  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB  
- **Authentication:** Sessions / Cookies / JWT  
- **Tools:** Git, GitHub, VS Code
